# 可能有用的文件备份

此文件夹包含从项目根目录移除的测试、调试和可能重复的文件。

## 文件清单

### 调试脚本（.js 文件）
这些是用于调试数据库和测试功能的 Node.js 脚本：

- **check-events.js** - 查询数据库中的所有活动记录
- **check-orders.js** - 查询最近订单的创建时间信息
- **check-tables.js** - 查看数据库表结构
- **test-env.js** - 测试环境变量是否正确加载
- **test-filter.js** - 测试订单日期筛选逻辑
- **env-load.js**（原 prisma/env-load.js）- 测试环境变量加载

### HTTP 测试文件
用于测试 API 接口的 REST Client 测试文件：

- **api-test.http** - 完整的 API 接口测试集，包含各种测试场景
- **quick-diagnose-order.http** - 订单问题快速诊断工具

### 可能重复/不需要的文件

- **app/page.tsx** - 根路径页面，与 `/events` 页面功能重复
- **prisma.config.ts** - Prisma 配置文件，现代项目通常不需要

## 如何使用这些文件

所有文件都已添加了注释，说明了它们的原始位置和用途。

如果需要恢复某个文件，只需将其移回原来的位置即可。

## 可以安全删除吗？

这些文件主要用于开发调试，生产环境不需要。如果：

1. 项目运行正常
2. 不需要进行底层调试
3. 确认不会再使用这些测试脚本

那么可以安全删除整个"可能有用"文件夹。
