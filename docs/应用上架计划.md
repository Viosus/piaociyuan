# 票次元 中国市场上架计划

## 一、行政审批办理

### 1.1 ICP 备案（网站备案）

- **入口**：阿里云控制台 → ICP 备案 → https://beian.aliyun.com/
- **所需材料**：
  - 企业营业执照扫描件
  - 法人身份证正反面
  - 网站负责人身份证 + 手持照
  - 域名证书（万网导出）
  - 备案真实性核验单
- **操作步骤**：
  1. 登录阿里云备案系统，填写主体信息（企业名称、证件号、法人信息）
  2. 填写网站信息（域名、网站名称"票次元"、网站用途说明）
  3. 上传主体证件和网站负责人证件
  4. 完成人脸核验（阿里云APP扫码）
  5. 提交初审，等待阿里云审核（1-2个工作日）
  6. 初审通过后提交管局审核
  7. 管局审核通过，获得ICP备案号
- **注意**：
  - 域名实名认证须与备案主体一致
  - 备案期间可开发不能正式运营
  - 通过后须在网站底部展示备案号并链接 beian.miit.gov.cn
- **预计时间**：2-4 周

### 1.2 APP 备案

- 2023年8月起工信部要求所有APP必须备案
- **前置条件**：ICP备案完成
- **所需材料**：
  - ICP备案号
  - APP名称（票次元）
  - 包名 `com.piaociyuan.mobile`
  - APP公钥
  - SHA-256签名指纹 / Bundle ID
  - 功能简介
- 通过阿里云APP备案入口提交
- **预计时间**：ICP完成后1-2周

### 1.3 软件著作权（软著）— 代理加急

- 国内安卓商店硬性要求，iOS不需要
- 委托代理（费用约500-1500元）
- **提供材料**：
  - 营业执照复印件盖章
  - 法人身份证
  - 源代码前30页 + 后30页（每页 ≥ 50行，去注释空行）
  - 软件说明书10-15页
- **软件名称**："票次元票务管理系统 V1.0"
- **预计时间**：15-30工作日

### 1.4 时间线

```
立即开始：ICP备案 + 联系软著代理
第1-3周：等待ICP审核（同步完成代码改造）
第3周：ICP通过 → 提交APP备案 + 网站正式上线
第3-5周：等待APP备案 + 软著出证
第4周：先提交iOS App Store（不需要软著和APP备案）
第5-6周：所有资质到手 → 提交国内Android商店
```

## 二、应用商店上架

### 2.1 iOS — Apple App Store

- 不需要软著和APP备案即可提交
- **需要**：
  - Apple Developer 账号（企业）
  - 隐私政策URL
  - App描述截图
- **审核周期**：1-3天

### 2.2 Android — 国内应用商店

- **目标商店**：华为应用市场、小米应用商店、OPPO软件商店、vivo应用商店、应用宝
- **需要**：
  - 软著证书
  - APP备案号
  - 隐私政策URL
  - APK签名文件
- 各商店需分别注册开发者账号
- **优先级**：华为 > 小米 > 应用宝 > OPPO > vivo

### 2.3 不上架 Google Play

- 目标市场为中国大陆，Google Play 在国内不可用

---

## 三、合规要求

### 3.1 隐私政策

- 必须包含SDK清单（第三方SDK及收集的数据）
- 必须包含个人信息收集清单表格
- 首次使用必须弹出隐私同意弹窗，阻塞式（不同意不能使用）

### 3.2 内容合规

- 数字藏品功能标记为"即将上线"
- 去除所有NFT/区块链/OpenSea等国际化术语
- 保留3D模型展示功能

### 3.3 网站底部

- 展示ICP备案号并链接 beian.miit.gov.cn
- 如有公安备案也需展示
