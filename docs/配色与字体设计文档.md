## 🎨 设计理念

票次元采用**浅梦幻粉色**作为主色调，搭配**温暖米黄色**背景，营造年轻、温馨、浪漫且充满梦幻感的追星票务平台氛围。

### 核心设计原则

- **高对比度**: 确保在米黄色背景上的可读性
- **一致性**: 所有交互元素使用统一的粉色系
- **可访问性**: 遵循 WCAG 2.1 标准
- **视觉层次**: 通过透明度和色彩深浅建立清晰的视觉层次

## 🔧 使用指南

### 1. 如何使用主题色

#### 背景色
```tsx
// 主粉色背景
<div className="bg-[#FF69A6]">内容</div>

// 使用 Tailwind 类
<div className="bg-pink-500">内容</div>

// 半透明背景
<div className="bg-pink-500/10">内容</div>
```

#### 文字颜色
```tsx
// 主粉色文字
<p className="text-pink-500">文本</p>

// 白色文字 (用于粉色背景上)
<p className="text-white">文本</p>

// 深色文字 (用于浅色背景上)
<p className="text-gray-900">文本</p>
```

#### 边框颜色
```tsx
// 粉色边框
<div className="border border-pink-300">内容</div>

// 悬停时改变边框
<div className="border border-pink-300 hover:border-pink-500">内容</div>
```

### 2. 响应式字体

```tsx
// 移动端小字，桌面端大字
<h1 className="text-xl md:text-2xl lg:text-3xl">标题</h1>
```

### 3. 深浅模式 (可选扩展)

虽然当前使用明亮主题，但预留了深色模式支持：

```css
/* 在 globals.css 中可以添加 */
@media (prefers-color-scheme: dark) {
  :root {
    --background: #0f0f14;
    --foreground: #ffffff;
  }
}
```

### 4. 动画与过渡

```tsx
// 标准过渡
<button className="transition-all duration-300">按钮</button>

// 悬停缩放
<div className="hover:scale-105 transition-transform">卡片</div>

// 透明度过渡
<div className="opacity-0 group-hover:opacity-100 transition-opacity">内容</div>
```

## 📊 可访问性考虑

### 对比度检查

| 组合 | 对比度 | WCAG 等级 |
|------|--------|----------|
| `#FF69A6` 背景 + 白色文字 | 3.2:1 | AA (大字) |
| `#EAF353` 背景 + 黑色文字 | 14.5:1 | AAA |
| 粉色按钮 + 白色文字 | 3.2:1 | AA (大字) |

### 建议

1. **重要文本使用高对比度**: 在粉色背景上使用白色文字
2. **提供焦点指示**: 所有交互元素需要明显的焦点状态
3. **避免仅用颜色传达信息**: 使用图标 + 文字 + 颜色的组合

```tsx
// ✅ 好的示例
<button className="bg-pink-500 text-white focus:ring-2 focus:ring-pink-300">
  ✓ 提交订单
</button>

// ❌ 避免
<button className="bg-pink-500/30 text-pink-400">提交</button>
```
