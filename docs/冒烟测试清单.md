## 出现问题时的回滚

每个 commit 都是独立的可 revert 单元：
```bash
# 例如 bcrypt 出问题：
git revert ddcb80b   # 登录系统改造
git push

# Mobile typecheck 修复出问题：
git revert de43790
git push

# 全部回滚：
git reset --hard beb55fb && git push --force-with-lease  # 危险，先和团队确认
```

不要 `--force` push 到 main 没和谁确认就推。
